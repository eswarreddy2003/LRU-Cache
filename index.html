LRU (Least Recently Used) Cache is a caching strategy that organizes items based on their usage. When the cache reaches its capacity limit, it evicts the least recently accessed item first.

This visualizer demonstrates the LRU algorithm with a doubly-linked list and hash map implementation, showing real-time updates as you interact with the cache.

  • Real-time visualization of cache operations
  • Detailed performance metrics tracking
  • Interactive tutorial mode for learning
  • Responsive design works on all devices
  • Action history log for debugging
